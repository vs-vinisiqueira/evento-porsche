# VIP by Porsche

Site institucional fictício desenvolvido para um projeto acadêmico, com o objetivo de divulgar o evento **VIP by Porsche**, uma experiência presencial premium voltada para clientes de alto nível da marca Porsche.

## Sobre o projeto

O **VIP by Porsche** é um evento fictício criado para representar uma experiência exclusiva envolvendo luxo automotivo, tecnologia, inovação, networking e a apresentação conceitual de um novo carro ainda não anunciado.

O site foi desenvolvido com foco em uma identidade visual sofisticada, utilizando uma estética minimalista baseada em tons escuros, detalhes dourados e elementos visuais inspirados no universo premium da Porsche.

## Objetivo do site

O objetivo principal do site é apresentar as informações essenciais do evento e incentivar os convidados a solicitarem inscrição.

O site contém seções como:

- Página inicial com chamada principal;
- Apresentação do evento;
- Experiências e destaques;
- Programação;
- Formulário de inscrição;
- Rodapé institucional.

## Tecnologias utilizadas

- HTML5
- CSS3

## Identidade visual

A identidade visual do projeto foi construída com base em:

- Preto absoluto como cor dominante;
- Dourado metálico para detalhes, títulos e destaques;
- Branco para contraste e leitura;
- Cinza metálico como cor de apoio;
- Layout minimalista, moderno e sofisticado.
